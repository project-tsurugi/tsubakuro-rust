# README.mdについて

tsubakuro-rust-pythonにはREADME.mdが2つあるが、以下のような使い分けである。

- tsubakuro-rust-python/README.md
  - GitHubで公開する目的のREADME。
  - 開発者向けの情報（ビルド方法等）も記載する。
- tsubakuro-rust-python/docs/README.md
  - PyPIで公開する目的のREADME。
    - PyPIにはこのREADMEをアップロードする。
  - 利用者向けの情報しか記載しない。
    - ビルド方法等は記載しない。それ以外はGitHub向けのREADMEと同様の内容。
